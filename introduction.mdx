## Overview

am provides your default documentation space. You can organize pages, collaborate with teams, and maintain project knowledge effortlessly. Customize layouts and themes to match your brand while keeping everything searchable and up-to-date.

<Callout kind="info">
  am integrates seamlessly with your workflow, supporting Markdown, MDX, and version history for all docs.
</Callout>

## Quick Start

Get your first documentation space running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com` with your email.
  </Step>
  
  <Step title="New Space" icon="plus">
    Click "New Space" and name it for your project, like "My Project Docs".
  </Step>
  
  <Step title="Add Page" icon="file-plus">
    Create your first page using the editor. Start with Markdown:
    
    ````markdown
    # Welcome
    
    This is your first doc page.
    
    - List items
    - Code blocks
    ```
    console.log("Hello, am!");
    ```
    ```
    ````
  </Step>
  
  <Step title="Publish" icon="globe">
    Save and publish. Share the link: `https://docs.example.com/my-space/welcome`.
  </Step>
</Steps>

<CodeGroup tabs="Markdown,MDX">
  ````markdown
  ## Simple Markdown
  
  Use headings, lists, and code blocks for basic docs.
  ```
  ````
  
  ````mdx
  ## MDX with Components
  
  <Callout kind="tip">Enhance with interactive elements.</Callout>
  ````

</CodeGroup>

## Brand Guidelines

Maintain consistency with am's branding. Use the primary color `#3B82F6` for accents, buttons, and links. Pair it with neutral backgrounds for readability.

| Element       | Color Value    | Usage Example              |
|---------------|----------------|----------------------------|
| Primary      | `#3B82F6`     | Buttons, links, highlights |
| Secondary    | `#1E40AF`     | Hover states, borders      |
| Background   | `#FFFFFF`     | Page backgrounds           |
| Text         | `#1F2937`     | Body text                  |

<Callout kind="tip">
  Customize themes in space settings to apply `#3B82F6` automatically.
</Callout>
